Hi Bruno,

When creating new branches it is better to use the maven-release-plugin's
"branch" goal:

https://maven.apache.org/maven-release/maven-release-plugin/branch-mojo.html

This goal automagically updates the scm tags in each of the pom files.
Failure to update the scm tags causes chaos when the new branch is
released, as you may have found out already judging by all the subsequent
emails (which I haven't looked at yet in detail).

Matt
