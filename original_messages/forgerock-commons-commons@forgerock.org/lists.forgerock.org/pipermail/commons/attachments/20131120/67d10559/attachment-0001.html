+  &lt;section name=&quot;About OpenDJ Contact Manager&quot;&gt;
+
+   &lt;p&gt;
+    &lt;a href=&quot;http://forgerock.com/products/open-identity-stack/opendj/&quot;
+    &gt;OpenDJ directory services&lt;/a&gt; give modern mobile applications easy access
+    to directory data through a
+    &lt;a href=&quot;http://commons.forgerock.org/forgerock-rest/&quot;
+    &gt;ForgeRock common REST&lt;/a&gt; interface.
+
+    OpenDJ Contact Manager is an Android application
+    that demonstrates use of OpenDJ directory server's REST interface
+    to search for and to read user resources.
+
+    When you retrieve the resource for a user from OpenDJ directory server,
+    OpenDJ Contact Manager lets you do the following:
+   &lt;/p&gt;
+
+   &lt;ul&gt;
+    &lt;li&gt;Add the user to your Android address book.&lt;/li&gt;
+    &lt;li&gt;Place a call to the user.&lt;/li&gt;
+    &lt;li&gt;Send email to the user.&lt;/li&gt;
+    &lt;li&gt;Geolocate the user's address.&lt;/li&gt;
+    &lt;li&gt;Get the resource for the user's manager.&lt;/li&gt;
+   &lt;/ul&gt;
+
+   &lt;p&gt;
+    The directory data itself is exposed as REST resources of HTTP
+    using the directory HTTP connection handler,
+    with a mapping from LDAP entries to REST resources
+    configured in a file called &lt;code&gt;http-config.json&lt;/code&gt;.
+
+    For details about the OpenDJ REST interface
+    and for examples showing how to use it,
+    see the &lt;i&gt;OpenDJ Administration Guide chapter&lt;/i&gt;,
+    &lt;i&gt;&lt;a href=&quot;http://docs.forgerock.org/en/opendj/2.6.0/admin-guide/#chap-rest-operations&quot;
+    &gt;Performing RESTful Operations&lt;/a&gt;&lt;/i&gt;.
+   &lt;/p&gt;
+
