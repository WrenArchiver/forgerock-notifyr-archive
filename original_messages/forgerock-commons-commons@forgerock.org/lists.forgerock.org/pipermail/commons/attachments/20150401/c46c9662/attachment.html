                                            <div class="preformatted panel" style="border-width: 1px;; border: 1px solid #ccc; background: #f5f5f5; font-size: 12px; line-height: 1.333; font-family: monospace; border: 1px solid #ccc; -moz-border-radius: 3px 3px 3px 3px; border-radius: 3px 3px 3px 3px; margin: 9px 0">

                                                </li> 
                                                <li>
                                                    the resource server receives a request from the proxy with the URI set to &quot;resource server&quot; and the Host set to &quot;proxy&quot;
                                                </li> 
                                                <li>
                                                    the resource server, for whatever reason, needs to send back an <b>accessible</b> URL, e.g. a Location header in a redirect. The URL must be formed from the Host header since the host contained in the URL cannot be accessed directly by the user agent.
