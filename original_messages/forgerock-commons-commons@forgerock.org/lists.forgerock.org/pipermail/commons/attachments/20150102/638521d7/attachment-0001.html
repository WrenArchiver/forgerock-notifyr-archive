<h3>Log Message</h3>
<pre>Implemented search filters.
- The search filters are available in the main activity, in the action bar menu.
At this point, 4 filters have been implemented to allow an exact match search, 
a search by employee type (or function) such as Payroll, etc. 
and a search by manager to retrieve all employees who have the same manager.
The filter selection is stored in user preference and is loaded at application startup.</pre>
