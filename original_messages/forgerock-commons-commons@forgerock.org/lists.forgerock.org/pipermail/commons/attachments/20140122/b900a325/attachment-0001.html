<h3>Log Message</h3>
<pre>* Add support for logging at trace level without localized message

* Add implementation of SLF4J Marker interface that contains the message to log
 as a LocalizableMessage : new LocalizedMarker class

* Add automatically a LocalizedMarker containing the LocalizableMessage to all 
 logging methods that are not called with a marker 
 This allow to retrieve the LocalizableMessage when implementing a custom SLF4J 
 Logger Adapter.

This is needed for OPENDJ-1288 : Migrate I18n and logging support to i18n framework and SLF4J</pre>
