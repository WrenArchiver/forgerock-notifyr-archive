<h3>Log Message</h3>
<pre>CHF-26 (CMN-191) Move `HttpClientHandler` in the `handler` package

Note that this doesn't make sense to hide that class since it's hosting
all of the `Options` used for configuration. I did not added a factory
method in `Handlers` as well because its important that the caller can
call the `close()` method on the handler, for that, it needs to be exposed
to the implementation class directly.</pre>
