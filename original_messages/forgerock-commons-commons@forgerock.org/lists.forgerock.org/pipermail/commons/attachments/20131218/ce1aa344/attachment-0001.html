<h3>Log Message</h3>
<pre>CR-2736 Fix for DOCS-121: Maven resource filtering should happen after common content is copied

This has an impact on projects using the plugin. Now the order needs to be like this:
 
             &lt;execution&gt;
             &lt;id&gt;copy-common&lt;/id&gt;
             &lt;phase&gt;pre-site&lt;/phase&gt;
             &lt;goals&gt;
              &lt;goal&gt;boilerplate&lt;/goal&gt;
             &lt;/goals&gt;
            &lt;/execution&gt;
            &lt;execution&gt;
             &lt;id&gt;filter-sources&lt;/id&gt;
             &lt;phase&gt;pre-site&lt;/phase&gt;
             &lt;goals&gt;
              &lt;goal&gt;filter&lt;/goal&gt;
             &lt;/goals&gt;
            &lt;/execution&gt;</pre>
