<h3>Log Message</h3>
<pre>CREST-109 (CR-3482)
Support additional parameters on all request types.
Additionally, for consistency, switch the ActionFunction optional parameter list from

    action(String resourceName, [String actionId,] Map params, Map content[, List fieldFilter][,Map context])

to

    action(String resourceName, [String actionId,] Map content, Map params[, List fieldFilter][,Map context])</pre>
