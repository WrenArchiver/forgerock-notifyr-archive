<h3>Log Message</h3>
<pre>CR-5817 DOCS-188 Remove JavaScript icons [+][-] for (un)folding &lt;screen&gt;

Some time ago, we were including &lt;screen&gt; listings
without continuation characters for long lines,
making it hard to copy &amp; paste.

So we added a JavaScript function linked to an icon in the HTML
that would unfold &lt;screen&gt; content, making it easier to copy &amp; paste. 

But a better solution was to add continuation characters for long lines,
which we have done throughout the docs since then.
Formatting no longer needs to change in order to copy &amp; paste. 

As attaching this functionality to an icon added by JavaScript
not only is no longer necessary 
but also probably does not help accessibility,
this patch removes the functionality. 

As a side effect, this should fix DOCS-150 as well.</pre>
