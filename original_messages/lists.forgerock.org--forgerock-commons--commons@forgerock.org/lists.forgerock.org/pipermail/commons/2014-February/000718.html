Hi,

For some time we have been missing web sites for a number of commons
projects. That's unfortunate because there's quite a bit of good stuff in
commons.

For DOCS-104 <https://bugster.forgerock.org/jira/browse/DOCS-104>, with
lots of review help from Mike Jang, I've added a top-level
http://commons.forgerock.org/ that has links to web sites for current
commons projects.

Many of the sites are skimpy. It would be nice to continue to add to them
for active projects including short explanations, samples showing how to
use libraries, and so forth.

For an example of a commons project site with some handy docs, see
http://commons.forgerock.org/forgerock-rest/json-resource-servlet/

If you feel like writing in HTML-like format, you can put your docs under
src/site/xdoc/ and link to them from src/site/site.xml. Or you can use
Markdown for example as for
http://commons.forgerock.org/samples/mobile/phonegap/. (Or APT, FML... See
https://maven.apache.org/plugins/maven-site-plugin/examples/creating-content.html
.)

Build jobs can be found under https://builds.forgerock.org/view/Docs/.

Regards,
