<pre>OPENDJ-1575 Menu is not accessible
As some phones don't possess the menu button and the fix is to provide
the action bar (the 3-dot on top).
- Moved up minimal API from 8 to 14.
- Added action bar and icons.
- Removed dependency in pom file.</pre>
