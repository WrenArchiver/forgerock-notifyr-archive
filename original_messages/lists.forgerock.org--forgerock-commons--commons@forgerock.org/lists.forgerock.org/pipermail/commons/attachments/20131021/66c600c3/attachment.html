        <p style='margin-top:0;margin-bottom:10px;'>The Request contract for getResourceName and getResourceNameObject states:</p>

{blockquote}<br/>
Returns the non-{@code null} name of the JSON resource to which this request should be targeted.{blockquote}

<p style='margin-top:0;margin-bottom:10px;'>The newXxxRequest factory methods in Requests will aptly throw NPE on a null resource name, but the setResourceName(ResourceName resourceName) method allows nulls which will break this contract.  See this unit test patch.</p>
