<h3>Log Message</h3>
<pre>CR-7019 DOCS-296 Include support for DocBook profiling

This patch adds support for conditional text in the style of DocBook profiling.
Profiling is described in Bob Stayton's book on using DocBook XSL,
http://www.sagehill.net/docbookxsl/Profiling.html.

Thanks to Chris Lee for review.</pre>
