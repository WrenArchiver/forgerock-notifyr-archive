<h3>Log Message</h3>
<pre>CREST-133: Migrate CREST from Futures to Promises

Apply changes from review comments:

* added @FunctionalInterface as comment to each callback interface
* changed for loop to while
* renamed waitForResult() to await() since completion can occur due to exceptions/cancellation as well as results
* refactored timed getOrThrowUninterruptibly(): push interrupt handling into timed await() so that we only have to deal with timeout handling in one place and also allowing us to take advantage optimised fast-path.</pre>
