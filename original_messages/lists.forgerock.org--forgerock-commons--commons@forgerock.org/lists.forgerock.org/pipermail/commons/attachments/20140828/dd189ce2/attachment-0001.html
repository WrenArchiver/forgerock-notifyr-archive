<h3>Log Message</h3>
<pre>CREST-170, CR-4313 - CREST API and router should be configurable to add a warning and fail header on response when request does not contain version. This patch allows for the addition of AdviceContexts to be added to the Context chain, to which Advice (strings) can be added. In this patch, these Advices are then read out for the HTTP CREST implementation via RequestRunner, which translates them into HTTP Headers. However, the Advice/AdviceContext themselves are not HTTP-specific.</pre>
