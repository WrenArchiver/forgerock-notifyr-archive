9 + ### Access
10 + The access audit topic logs system boundary events. For example, the initial request and final response for a given 
11 + request to the system being audited.
12 +
13 + ### Activity
14 + The activity audit topic logs audit event on resources. For example, if the system being audited had a user resource, 
15 + operations on that user resource would be audited.
16 +
17 + ### Authentication
18 + The authentication audit topic logs authentication attempts and their success or failure. This event can also log 
19 + logout events.
20 +
21 + #### CAF Authentication Logging

23 + must direct log data to the commons audit authentication endpoint.
24 +
25 + ### Configuration
26 + The config audit topic logs operations on the systems configuration.
