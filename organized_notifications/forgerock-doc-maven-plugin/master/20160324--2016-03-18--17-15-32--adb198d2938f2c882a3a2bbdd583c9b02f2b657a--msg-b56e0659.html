adb198d2938 DOCS-398 Enable link checking for default lifecycle

This patch introduces a new boolean option, `<forceEarlyLinkCheck>`.
When `true` (default: `false`), `docbook-linktester` runs just before building docs,
rather than during the `site` goal.

This can be useful when the `site` goal is never run, for example for a release build.		 18 Mar 2016
