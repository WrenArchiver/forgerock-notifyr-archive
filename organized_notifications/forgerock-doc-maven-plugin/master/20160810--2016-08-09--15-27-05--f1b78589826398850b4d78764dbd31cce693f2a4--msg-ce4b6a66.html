50 +
51 + The issue is not completely fixed within the documentation tools.
52 + You can work around the issue, however, with the following steps:
53 +
54 + 1.  Build with `<verbose>true</verbose>` or `-Dverbose=true`.
55 +     For release profiles, you must use the former in the plugin configuration.
56 +     The Maven release plugin does not pass system properties
57 +     to the other plugins that it executes.
58 + 2.   Use the Jenkins Text Finder plugin to scan both the linktester report file
59 +     and the console output for the job for `^.*ERROR.*$`,
60 +     and flag the build and send notification if a match is found.
