88d487b2e37 DOCS-355 Different expectations for shared/images/ between HTML and PDF

This patch edits built HTML so that source files can refer to `../shared/images/`.

References to `shared/images/` will no longer work in HTML-based output.

Tested with `forgerock-doc-maven-plugin` and `mvn clean -N site -Dformats=bootstrap,pdf,html,webhelp`.
