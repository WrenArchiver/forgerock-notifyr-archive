f45098652f4 DOCS-413 Create a HTML cover page for PR jobs

This patch adds an unstyled HTML cover page
pointing to the Bootstrap docs under `site/doc/`.

The layout of Bootstrap docs under `target/site/doc/`
ensures that the links to PDFs from Bootstrap docs work,
whereas the same links under `target/docbkx/bootstrap` are broken.

This patch also deprecates the class for adding a redirect page,
`target/site/doc/index.html`, as that page is no longer useful
now that draft docs have moved to the community site.		 Mark 02 May - 08:09 AM
