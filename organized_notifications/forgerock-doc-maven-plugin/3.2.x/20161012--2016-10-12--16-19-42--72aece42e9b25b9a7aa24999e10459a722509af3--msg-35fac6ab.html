72aece42e9b DOCS-425 Clean up table attributes during preprocessing

This adds a filter to remove attributes from table and informaltable elements
that cause rules to appear in the output.

It would be particularly useful with the Asciidoc produced by
the current implementation of the CREST API descriptor,
but also results in more uniform formatting for handwritten tables.		 Mark 12 Oct - 04:19 PM
