e41916e5975 CUI-93 Introduce ESLint configuration file

We plan to transition from JSLint to ESLint due to it's more
modular and configurable nature. The first step in that transition
is to introduce an ESLint configuration file that provides an
equivalent set of checks to the version of JSLint we use. This
configuration file can be used immediately by developers by
configuring their IDEs to use ESLint.

I've have included a few warning rules to demonstrate their use.		 11 Sep 2015
