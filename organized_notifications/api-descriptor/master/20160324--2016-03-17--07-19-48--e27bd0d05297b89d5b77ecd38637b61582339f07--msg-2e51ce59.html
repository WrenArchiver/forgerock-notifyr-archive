e27bd0d0529 RAPID-69 Demonstrate that the descriptor would work for OpenIG 4

This patch adds examples showing descriptors for a simple OpenIG configuration:

* A single route exposes monitoring statistics.
* A single UMA RS configuration allows management of UMA shares.

This patch also includes experiments that do not conform to the current specification:

* An alternative that shows services separately, not inline in the paths object.
* An example that shows services with inlined sub resource endpoints.

A number of APIs, in particular in OpenIG, have statically defined services,
not only statically defined resources. But the paths depend entirely on the configuration.		 17 Mar 2016
