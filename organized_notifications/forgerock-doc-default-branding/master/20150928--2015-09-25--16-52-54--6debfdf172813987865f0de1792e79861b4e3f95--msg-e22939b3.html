1 + # ForgeRock Docs Default Branding
2 +
3 + By default, ForgeRock draft documentation uses the branding from this project.
4 +
5 + The branding project contains CSS for in-progress and released docs,
6 + and custom DocBook stylesheets for supported output formats.
7 +
8 + In order to use the default branding, get a common content artifact
9 + by using the `maven-dependency-plugin` goal, `unpack`.
10 +
11 + The `forgerock-doc-maven-plugin` depends on this project.
